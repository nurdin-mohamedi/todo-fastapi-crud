# Todo Application with FastAPI

1. Download and install [uv library](https://docs.astral.sh/uv/getting-started/installation/).
2. After making sure that uv is installed successfully, clone this repo.
3. Inside the clone folder, run the command `uv sync`.
4. Activate the virtual environment in your terminal session, run
5. You can perform unit tests by running `pytest` in an activated virtual environment.
6. You can run a local development server by running `fastapi dev main.py`.
7. Test the API by visiting <http://localhost:8000/docs>
